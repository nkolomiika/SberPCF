# Заметки проекта

Заметки — это общие материалы проекта: методология, результаты разведки, черновики находок и любые совместные записи. Заметки живут во вкладке **Notes** проекта и отображаются списком карточек.

## Как работать с заметками

**Создать заметку:**
1. Откройте проект → вкладка **Notes**.
2. Нажмите кнопку создания заметки.
3. Введите заголовок и содержимое, сохраните.

Текст заметок пишется в **Markdown** — редактор форматирует его по ходу набора. В заметку можно вставлять изображения.

**Редактировать / удалить.** Открыв свою заметку, воспользуйтесь соответствующими действиями.

> Редактировать и удалять можно **только собственные** заметки. Чужие заметки открываются в режиме просмотра.
