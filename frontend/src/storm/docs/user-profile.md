# Профиль и безопасность

В разделе **Profile** вы управляете своей учётной записью. Он открывается из меню по клику на ваше имя в правом верхнем углу. Профиль разбит на вкладки: **Account**, **Security** и **API & Automation**.

## Account — данные профиля

На вкладке **Account** можно загрузить **аватар** (PNG/JPEG/WEBP/GIF, до 5 МБ) — он отображается рядом с вашим именем в интерфейсе и в списках участников.

Карточка идентичности (логин, email, роль, ID) показана **только для чтения** — этими данными управляет администратор.

## Security — двухфакторная аутентификация

На вкладке **Security** настраивается **двухфакторная аутентификация (2FA)** по одноразовым кодам (TOTP, приложение-аутентификатор):

- **Включение** — Storm показывает QR-код (и секрет для ручного ввода); отсканируйте его в приложении и подтвердите 6-значным кодом.
- **Отключение** — потребуется ввести пароль от учётной записи; все активные сессии будут завершены.

После включения при каждом входе, помимо логина и пароля, потребуется код из приложения.

## API & Automation — ключи для автоматизации

Вкладка **API & Automation** (раздел **API keys**) предназначена для токенов, которыми пользуются боты, CI и внешние агенты — без интерактивного входа. Это машинный доступ к API Storm.

> Выпустить токен может **любой пользователь** — но только в рамках своих прав (доступ только к вашим проектам). Как создавать токены, задавать права (scopes), доступ к проектам и срок действия — в документе **Agent API**.
