# Начало работы

Storm — платформа для командной работы над проектами по анализу защищённости: вы ведёте в ней хосты и сервисы цели, заводите найденные уязвимости, храните заметки и в конце выгружаете отчёт в Word.

Эта глава поможет войти в систему и сориентироваться в интерфейсе.

## Как получить доступ

Учётные записи создаёт администратор, приглашая по email — самостоятельной регистрации нет. Вы получаете **письмо со ссылкой-приглашением**:

1. Перейдите по ссылке из письма.
2. Придумайте **логин** (имя пользователя) — система сразу проверит, свободен ли он.
3. Задайте **пароль**.
4. Подтвердите — учётная запись активирована, можно входить.

## Вход в систему

1. Откройте страницу входа Storm.
2. Введите **логин** и **пароль**, которые задали при активации.
3. Если для вашей учётной записи включена двухфакторная аутентификация, дальше система попросит **6-значный код** из приложения-аутентификатора.

Сессия держится на защищённых cookie и продлевается автоматически, пока вы работаете. Если вы долго отсутствовали, система может попросить войти снова.

## Обзор интерфейса

После входа открывается рабочая область. Слева — **боковое меню (сайдбар)** с разделами:

- **Projects** — список проектов, доступных вам. Это главный экран, с которого начинается работа.
- **Tasks** и **My Tasks** — разделы задач (в разработке).
- **Docs** — эта документация.
- **Members** — управление пользователями рабочего пространства. Пункт виден **только администраторам**.

В правом верхнем углу — иконка уведомлений (колокольчик) и меню профиля: по клику на имя открывается переход в **Profile** и кнопка выхода (**Logout**).

Сайдбар можно свернуть кнопкой со стрелками в его шапке.
