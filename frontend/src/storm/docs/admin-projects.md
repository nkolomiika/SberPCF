# Проекты

Создание и удаление проектов — операции администратора. Обычные пользователи и лиды работают внутри уже созданных проектов, но не создают и не удаляют их.

## Создание проекта

1. Откройте раздел **Projects**.
2. Нажмите **New project**.
3. Укажите название, описание и, при необходимости, даты начала и окончания.
4. Сохраните.

После создания добавьте в проект участников (это может делать и лид/создатель) и, при необходимости, назначьте статус проекта.

## Редактирование карточки

Карточку проекта — название, описание, **статус** и даты — может править администратор, лид проекта и его создатель. Редактирование открывается кнопкой правки в строке проекта. Статусы проекта перечислены в главе «Проекты» документа **Пользователь**.

## Удаление проекта

Удаление проекта доступно **только администратору** (кнопка удаления в строке проекта, с подтверждением). Удаление необратимо и уносит связанные с проектом данные, поэтому выполняйте его осознанно.
