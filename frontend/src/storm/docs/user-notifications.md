# Уведомления

Уведомления — ваша личная лента событий в Storm. Список поводов намеренно узкий: система уведомляет только о том, что касается вас лично. Для истории «кто что сделал» есть лента активности проекта (см. главу «Проекты»).

## О чём приходят уведомления

- **Добавление в проект** — вас добавили участником в проект.
- **Смена статуса находки** — изменился статус уязвимости, которую **завели вы**.
- **Смена статуса проекта** — изменился статус проекта, в котором вы состоите.

## Где смотреть

Лента открывается по клику на **колокольчик** в шапке интерфейса; на нём — точка-индикатор непрочитанных. В самой панели показывается счётчик новых уведомлений.

**Действия:**
- клик по уведомлению открывает связанный объект и отмечает его прочитанным;
- кнопка **Mark all read** отмечает прочитанными все сразу;
- непрочитанные отмечены точкой.

Лента загружается при открытии приложения. Чтобы увидеть свежие уведомления, обновите страницу.
